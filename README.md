This project is a simple command-line calculator built using Python. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division. The calculator accepts user input, processes it, and displays the result in a clear and interactive way.
output---
--- Command Line Calculator ---
Enter expression: 5 + 3
Commands:
  clear  -> reset
  exit   -> quit

>>> 10 + 5
Result: 15.0

>>> 8 / 2
Result: 4.0

>>> 5 / 0
Error: Cannot divide by zero!

>>> hello
Error: Input must be in format: number operator number

>>> clear
Calculator cleared!

>>> exit
Goodbye!
>>> Features
>>> Supports arithmetic operations: +, -, *, /
Accepts user input in simple format (e.g., 5 + 3)
Input validation for correct format and numbers
Handles errors like invalid input and division by zero
Modular design using separate functions
Continuous execution using loop
Special commands:
clear → reset calculator
exit → terminate programming
